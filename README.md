# JavaDirectoryWalker
A simple Java helper class that makes it easy to traverse and work with filesystem. This class helps to get informations about files and directories in a filesystem. Feel free to fork this repository.

With this class you can:
* Navigate through directories
* Get directory informations
* Get subdirectories
* Get all files in directories and subdirectories
* Get folder size
and many more.

# Requirements
This class requires Java 17+ to work
